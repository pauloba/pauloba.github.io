Dear tech recruiter:

If you send me a message with a job offer and you expect me to invest my time answering, please read the following guidelines.

The job offer must include:

  • The name and description of the company, with real links to their website.
  • An accurate description of the possition, with the responsabilities to be acquired and the specific profile desired, not a list of "buzzwords" taken from Linkedin.
    • If you are looking for expert ninjas/rockstars in PHP, Python, Ruby, Java, C#, Angular, NodeJS, Bash, Photoshop, Illustrator, CSS, iOS, Android, BigData, AWS, Azure and network configuration: I'm not interested.
  • The salary, or at least a salary range ( avoid non-sense ranges like "between 20k and 60k" ).
  • Location: on site or remote. If remote is negotiable, please mention it.

These are guidelines based on basic netiquette, inspired by this message from a Python maillist.

Querido/a recruiter técnico:

Si me envías un mensaje con una oferta de empleo y esperas que invierta mi tiempo en responder, por favor lee las siguientes directrices.

La oferta debe incluír:

  • La descripción y el nombre de la empresa, con enlaces reales a la misma.
  • Una descripción precisa del puesto de trabajo, con las responsabilidades a adquirir y el perfil concreto deseado, no un listado de "buzzwords" sacadas de Linkedin.
    • Si buscas ninjas/rockstars expertos en PHP, Python, Ruby, Java, C#, Angular, NodeJS, Bash, Photoshop, Illustrator, CSS, iOS, Android, BigData, AWS, Azure y configuración de redes: no me interesa.
  • La oferta debe incluir sueldo, o al menos rango salarial ( con sentido, nada de "entre 20K y 60K" ).
  • Ubicación: on site o en remoto. Si el remoto es negociable, por favor menciónalo.

Estas son unas directrices basadas en nettiqueta básica, inspiradas por este mensaje de una lista de correo de Python.
